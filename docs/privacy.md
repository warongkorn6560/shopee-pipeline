# Privacy Policy — ShopeePipeline

**Last updated: 2026-06-20**

ShopeePipeline is a personal automation tool that generates and posts affiliate product videos to TikTok.

## Data We Collect

- TikTok OAuth tokens (access token and refresh token) to authenticate with the Content Posting API.
- No personal data from TikTok users is collected, stored, or shared.

## How We Use Data

- OAuth tokens are stored as encrypted GitHub Actions secrets and are used solely to post videos to the authorised TikTok account.
- Tokens are never shared with third parties.

## Data Retention

- Tokens are retained only as long as the TikTok integration is active.
- You can revoke access at any time from TikTok's Settings → Security → Connected apps.

## Third-Party Services

This app uses:
- TikTok Content Posting API v2
- Google Gemini (script generation)
- Microsoft Azure TTS (voiceover)
- fal.ai (video generation)

Each service has its own privacy policy.
