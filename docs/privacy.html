Privacy Policy — ShopeePipeline

Last updated: 2026-06-22

ShopeePipeline is an affiliate marketing automation platform that generates AI-powered short-form product videos and publishes them to social media platforms including TikTok, Instagram, and Telegram on behalf of the authorized account owner.

Information We Collect

  • TikTok OAuth tokens (access token and refresh token) obtained via TikTok's official OAuth 2.0 authorization flow, used solely to publish videos to the authorized TikTok account via the Content Posting API.
  • TikTok User ID obtained from the user info API, used only to route API calls to the correct account.
  • No data from TikTok followers, viewers, or third-party TikTok users is collected, processed, or stored.

How We Use TikTok Data

  • OAuth tokens are stored as encrypted secrets in GitHub Actions and are used exclusively to post AI-generated affiliate product videos to the authorized TikTok account.
  • We use the video.publish scope to upload and publish videos, and user.info.basic to verify the authenticated account.
  • No TikTok data is shared with third parties, sold, or used for any purpose beyond publishing videos for the account owner.
  • Videos published via this platform are AI-generated promotional content for Shopee affiliate products, compliant with TikTok's Community Guidelines and Advertising Policies.

Data Storage and Security

  • OAuth tokens are stored as encrypted GitHub Actions secrets and are never logged, displayed, or transmitted outside of GitHub's secure secret storage.
  • Tokens are rotated automatically and can be revoked at any time by the account owner.
  • No personal data is stored in databases, files, or third-party services.

Data Retention

OAuth tokens are retained only while the TikTok integration is active. You can revoke access at any time from TikTok Settings → Security → Connected apps → ShopeePipeline → Remove access.
